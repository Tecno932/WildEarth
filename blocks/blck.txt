| Archivo             | Rol principal                                                         |
| ------------------- | --------------------------------------------------------------------- |
| `Chunk.cs`          | Construye el mesh de un chunk (geometría y materiales).               |
| `WorldGenerator.cs` | Coordina la generación y visibilidad de chunks alrededor del jugador. |
| `ChunkManager.cs`   | Sistema alternativo de generación (redundante con WorldGenerator).    |
| `ChunkWorker.cs`    | Hilo que genera los datos de bloques con ruido.                       |                          |
| `ChunkStorage.cs`   | Cache y persistencia de datos binarios por chunk.                     |
| `AtlasBuilder.cs`   | Maneja el atlas de texturas y materiales.                             |
| `WorldSettings.cs`  | Maneja la semilla global del mundo.                                   |



| Mapa            | Rango         | Controla                                     |
| --------------- | ------------- | -------------------------------------------- |
| **Altura**      | 50 a 126      | El terreno base (montañas, llanuras, cuevas) |
| **Temperatura** | 0.0 a 1.0     | Qué tan cálido o frío es el bioma            |
| **Humedad**     | 0.0 a 1.0     | Qué tan verde o árido es                     |
| **Bioma**       | string o enum | Tipo (desierto, bosque, tundra, etc.)        |
